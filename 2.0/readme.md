# 占卜项目

## 理论依据
本项目模拟的占卜方法是蓍草卜筮法，语出自《周易・系辞上传》：
大衍之数五十，其用四十有九。分而为二以象两，挂一以象三， 揲之以四以象四时，归奇于A以象闰，五岁再闰，故再A而后挂。 天一地二，天三地四，天五地六，天七地八，天九地十。天数五，地数五，五位相得而各有合。天数二十有五，地数三十，凡天地之数五十有五。此所以成变化而行鬼神也。


## 简介
本项目实现了一个基于易经六十四卦和蓍草卜卦法的占卜工具，用户可以直接运行程序获取占卜结果。

## 部分特点
- 选择随机数生成作为占卜的模拟
- 随机数生成器和种子从rand换成了mt19937，更好地保证了随机性。
- 将占卜结果输出到一个文件中，便于后续查看和分析。

## 使用方法
1. 编译并运行`main.cpp`文件。
2. 程序会自动运行占卜过程，并将结果输出到`占卜结果.txt`文件中。

## 文件结构
- `divination.h`：定义了`Divination`类，包含了类的成员变量和成员函数的声明。
- `divination.cpp`：实现了`Divination`类的成员函数。
- `main.cpp`：包含了程序的入口点，创建了一个`Divination`对象并调用其`run()`函数。
